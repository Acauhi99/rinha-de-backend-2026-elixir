# Triage Labels

Mapeamento dos 5 papéis canônicos de triagem para labels reais deste repo.

| Label no skill | Label neste repo | Significado |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | Maintainer precisa avaliar issue |
| `needs-info` | `needs-info` | Aguardando mais info do reporter |
| `ready-for-agent` | `ready-for-agent` | Especificada, pronta para agente AFK |
| `ready-for-human` | `ready-for-human` | Precisa implementação humana |
| `wontfix` | `wontfix` | Não será acionada |

Quando skill mencionar papel de triagem, usar label correspondente desta tabela.
